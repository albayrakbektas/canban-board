# canban-boards

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Docker build
```
docker build -t canban-board:v1 .
```

### Docker deployment
```
docker run -p 80:8080 canban-board:v1
```
